## Problem

Нет единой точки входа для запуска типичных задач разработки (запуск приложения, форматирование кода, линтинг, тесты). Разработчикам приходится помнить конкретные команды или смотреть в документацию.

## Solution

Добавить `Makefile` в корень проекта со следующими целями:
- `make run` — запуск приложения
- `make fmt` — форматирование кода
- `make lint` — линтинг
- `make test` — запуск тестов
- `make help` — список доступных команд (опционально)
