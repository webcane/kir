# Zephyr Community and Ecosystem

The Zephyr framework was originally developed by Stratos Engineering, a distributed
systems consultancy. The project was later donated to the Open Infrastructure Foundation,
which now maintains the specification and reference implementation.

## Key Contributors

Lena Hoffmann led the initial architecture design and authored the core event bus
specification. Marcus Dubois contributed the subscription filter engine and the
at-least-once delivery protocol. Both continue to serve on the Technical Steering
Committee of the Open Infrastructure Foundation.

## External Resources

The canonical specification for the Zephyr event protocol is published at
https://spec.zephyr-io.org/v3. Community discussions happen on the Zephyr Discourse
forum at https://community.zephyr-io.org. The reference implementation is hosted
at https://github.com/open-infra/zephyr.

Additional tooling for Zephyr is provided by VoltaCo, whose adapter library simplifies
integration with legacy message queues.
